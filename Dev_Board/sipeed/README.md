## k210
- 400mhz
- 双核64位 risc-v
- 8mb SRAM，6mb通用，分4mb和2mb两个bank，还有2mb为kpu专用。
- 拥有kpu，apu
- 双核独立中断系统，7优先级，64个外部中断，可跨核触发
